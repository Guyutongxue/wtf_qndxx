# 去他妈的青年大学习

> 北京青年大学习一键学习

## 使用方法

1. 安装 Node，克隆本仓库，执行 `pnpm i` 安装依赖；
2. 新建 `.env` 文件，提供如下内容：  
   | 变量 | 值 |
   |---|---|
   | `USERNAME` | 青春北京手机号 |
   | `PASSWORD` | 青春北京密码（微信登录请重置）|
   | `ORG_ID` | 团组织 ID |
   | `TT_USERNAME` | TT 识图用户名 |
   | `TT_PASSWORD` | TT 识图密码 |
3. 执行 `pnpm start`。
